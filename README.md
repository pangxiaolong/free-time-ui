## 介绍

开发一点自己要到的vue组件，想到什么就写一个，不定时优化更新和增加新的玩意。
- AvatarWeather天气组件（带pop列表）

## 安装

###  通过 npm 安装 


```bash
# npm
npm i free-time-ui

npm i vue-jsonp
```
###  使用方式
main.js
```javascript
import FreeTimeUI from 'free-time-ui'
import VueJsonp from 'vue-jsonp'

Vue.use(FreeTimeUI)
Vue.use(VueJsonp)
```
view.vue
```html
<avatar-weather />
```
